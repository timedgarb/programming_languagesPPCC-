## Сравнение Паскаль и C/C++

### Таблица соответствия синтаксиса

| Паскаль | C | C++ | Пояснение |
|---------|---|-----|-----------|
| **Переменные и константы** |
| `Var A: integer` | `int A` | `int A` | Объявление целой переменной |
| `Var A: real` | `float A` | `double A` | Объявление вещественной переменной |
| `Var A: string` | `char A[100]` | `string A` | Объявление строки |
| `Const n = 2` | `#define n 2` | `const int n = 2` | Объявление константы |
| **Процедуры и функции** |
| `Procedure P(Var A: int)` | `void P(int *A)` | `void P(int &A)` | Передача параметра по ссылке |
| `Function f: real` | `float f()` | `double f()` | Функция без параметров |
| `Function f(a: int): int` | `int f(int a)` | `int f(int a)` | Функция с параметром |
| **Ввод/вывод** |
| `write('text')` | `printf("text")` | `cout << "text"` | Вывод на экран |
| `writeln('text')` | `printf("text\n")` | `cout << "text" << endl` | Вывод с переводом строки |
| `readln(x)` | `scanf("%d", &x)` | `cin >> x` | Ввод целого числа |
| `write(x:5)` | `printf("%5d", x)` | `cout << setw(5) << x` | Форматированный вывод |
| `write(r:5:2)` | `printf("%5.2f", r)` | `cout << fixed << setprecision(2) << setw(5) << r` | Вывод вещественных |
| **Операции** |
| `not x` | `!x` | `!x` | Логическое отрицание |
| `x xor y` | `x ^ y` | `x ^ y` | Побитовое исключающее ИЛИ |
| `x and y` | `x & y` | `x & y` | Побитовое И |
| `x or y` | `x \| y` | `x \| y` | Побитовое ИЛИ |
| `$F` | `0xF` | `0xF` | Шестнадцатеричная константа |
| **Структуры** |
| `type T = record ... end;` | `struct T { ... };` | `struct T { ... };` | Объявление структуры |
| `r.a := 5` | `r.a = 5` | `r.a = 5` | Доступ к полю структуры |
| **Массивы** |
| `array[1..10] of int` | `int v[11]` | `int v[11]` | Массив (индексы 0-10) |
| **Системные функции** |
| `clrscr` | `system("cls")` | `system("cls")` | Очистка экрана |
| Нет аналога | `system("pause")` | `system("pause")` | Пауза перед завершением |

## Требования

### Для компиляции C версии:
- Компилятор C (GCC, MinGW, Visual Studio)
- Поддержка стандарта C99 или выше

### Для компиляции C++ версии:
- Компилятор C++ (G++, MinGW, Visual Studio)
- Поддержка стандарта C++11 или выше

### Для работы русского языка:
- Операционная система Windows (для CP1251 кодировки)
- На Linux/Unix требуется изменить кодировку на UTF-8
